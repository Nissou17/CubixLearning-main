## 🚀 Installation et Utilisation

Pour accéder au site lancer le main.py, un invité de commande va apparaitre ou faite Ctrl + ù, écrivez "cd sources" puis "python main.py" et cliquer sur http://127.0.0.1:5000, le site va s'afficher et vous pourriez donc commencé par choisir l'explication du site dans l'assistant box en bas à gauche.
